# Rocha Selecao

Site conceito da Rocha Empreendimentos com foco em branding e apresentacao dos empreendimentos.

Os textos visiveis do HTML usam entidades numericas, evitando erros de acentuacao causados por editores do Windows.


## Identidade visual
A logo oficial enviada pelo cliente foi aplicada no cabeçalho, rodapé e favicon.
